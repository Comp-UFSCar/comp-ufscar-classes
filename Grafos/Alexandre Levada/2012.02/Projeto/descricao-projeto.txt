Projetos

Cada grupo deverá desenvolver uma "biblioteca" de funções básicas para grafos que deve conter implementações de 4 projetos da lista a seguir, sendo os 3 primeiros obrigatórios a todos e um cujo tema pode ser qualquer um de 4 a 9:

1) Obtenção de árvores spanning mínimas (algoritmos de Kruskal e Prim) (*)
2) Caminhos em grafos não ponderados (Busca em Largura e Profundidade) (*)
3) Caminhos em grafos ponderados (algoritmo de Djikstra para árvore de caminhos mínimos) (*)

4) Codificação de árvores com o código de Prüfer
5) O algoritmo de Floyd-Warshall
6) Algoritmo 2-Otimal para obtenção de circuitos Hamiltonianos
7) Algoritmos de Welsh & Powell para coloração de vértices
8) Algoritmo Húngaro para emparelhamentos em grafos bipartidos
9) Algoritmo Ford-Fulkerson para fluxo em redes
10) Geração de redes complexas (Aleatórias, Pequeno mundo e Livres de escala)

OBS: (*) projetos obrigatórios a todos os grupos

Para auxiliar na manipulação, visualização e plotagem dos grafos e resultados (árvores,por exemplo), podem ser utilizados diversos softwares disponíveis para esse fim dentre os quais destacam-se:

- NodeXL: plugin do Excel para visualização e extração de informações de grafos (Windows)
- Gephi: aplicativo open-source multiplataforma para visualização de grafos
- yEd Graph Editor: aplicativo para criação de representações gráficas e muito útil para conversão de formatos de arquivos de grafos (GML, GRAPHML, XGML, XLS)
- GraphStream: a dynamic graph library
- NetworkX: a Python language package for graph creation, manipulation and dynamics.
